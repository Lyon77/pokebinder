Cloud Sync

Sync your collection across devices using a private GitHub Gist.

Setup instructions
  1. Go to gist.github.com and create a secret gist with a file named collection.json (content can be {})
  2. Copy the Gist ID from the URL (the hash after your username)
  3. Go to GitHub Settings > Tokens and create a token with the gist scope
  4. Paste both values above and click Connect
